À propos SafePlaces Version 1.0

SafePlaces est une application mobile qui aide les étudiants à trouver rapidement des lieux importants : hôpitaux, commissariats, points Wi-Fi, bibliothèques, et autres endroits utiles.

L'application fonctionne même hors ligne grâce au stockage local, et permet d’enregistrer vos propres lieux personnalisés.

Elle exploite aussi des fonctionnalités du téléphone comme le GPS et l’appel téléphonique.

Géolocalisation : activée Appel téléphonique : activé Service externe : API OpenStreetMap Développement

L’application est développée avec :

Ionic 7 + Angular Capacitor (Android) Stockage local (Ionic Storage) GPS (Capacitor Geolocation) OpenStreetMap API (adresse → coordonnées) Développé par Dominique Nkoa-M1 ISOC – 2025
