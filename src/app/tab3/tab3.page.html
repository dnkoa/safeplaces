À propos SafePlaces SafePlaces est une application mobile éducative permettant d’enregistrer, modifier et localiser des lieux importants (écoles, hôpitaux, bibliothèques, points Wi-Fi, centres éducatifs, etc.) en se basant sur les fonctionnalités natives d’un smartphone Android. Fonctionnalités principales ✔ Enregistrement de lieux éducatifs ✔ Modification complète des informations d’un lieu ✔ Suppression d’un lieu ✔ Sélection automatique des coordonnées GPS (position actuelle) ✔ Distance calculée entre l’utilisateur et chaque lieu ✔ Affichage des lieux classés du plus proche au plus éloigné ✔ Stockage local (fonctionnement hors-ligne) ✔ Ouverture directe du lieu dans Google Maps Informations techniques Version : {{ version }} Dernière mise à jour : {{ lastUpdate }} Ionic + Angular (Standalone Components) Capacitor Geolocation Stockage local via StorageService
